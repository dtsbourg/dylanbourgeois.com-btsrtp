OASIS

A prey-predator project written in C(++) with a GUI user interface, as a freshman semester project in CS102. The entire implementation was custom-made, from the UI (restricted by precise guidelines) to the 'AI' of preys and predators. Decisions were made based on a variety of parameters like proiximity to food or danger, energy levels, location and surroundings, or number of surrounding peers. The idea of the simulation was to tinker with these parameters to observe which behaviours led to a stable ecosystem.

C

LOGISNAKE

An implementation of the classic game Snake, built on digital logic software Logisim. It was made using only simple logic gates, not a single line of code. The display was an 8x8 LED matrix, commanded by a custom I/O module. The game was playable on an FPGA board, using the provided buttons to navigate the snake.
